# Changelog

All notable changes to this project will be documented in this file.  The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/) and follows semantic versioning.

## [1.0.0] – 2026-06-09

### Added

- Initial launch kit for **SignalScan** including complete directory structure, documentation, website, tooling and sample report.
- Configurable brand, services and pricing JSON files.
- Responsive one‑page website with privacy, terms and ethical boundaries pages.
- Forms and intake templates.
- Legal and scope templates covering service scope, engagement terms, privacy notice, data retention and responsible disclosure.
- Report templates in HTML and Markdown, with a sanitised sample report.
- Python tooling for passive collection, engagement validation and report rendering, plus associated tests.
- Prospecting scorecards and research guides.
- Outreach scripts for multiple channels and scenarios.
- Delivery workflow documents and client folder template.
- Launch runbooks and deployment guides.
- Background documentation on category definition, SIGNAL Method, service boundaries, metrics and future roadmap.
